# TITAN BOND - Premium Industrial Adhesives

A modern, responsive website for TITAN BOND, a premium industrial adhesive company operating in Pakistan and formulated in the USA.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Premium Aesthetics** - Burgundy and gold color scheme matching product packaging
- **Product Showcase** - Display of industry products with professional imagery
- **Multiple Applications** - Highlights for footwear, industrial, furniture, automotive, and general use
- **Contact Information** - Easy access to phone, email, and location details
- **Smooth Navigation** - Seamless scrolling and intuitive navigation

## Deployment

### GitHub Pages
This website is ready to deploy on GitHub Pages:

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select `main` (or your branch) as the source
4. Save - your site will be live at `https://yourusername.github.io/titan/`

### Local Testing
Simply open `index.html` in your browser to view the website locally.
